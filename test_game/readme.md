building frotz for this project:
```
$ git clone https://github.com/danielricks/textplayer.git
$ cd textplayer
$ git clone https://github.com/DavidGriffith/frotz.git
$ cd frotz
$ make dumb
$ cd ..
```
setup the directory structure:
```
$ ./do_make_dir.sh

```
